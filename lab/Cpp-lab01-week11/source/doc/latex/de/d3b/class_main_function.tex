{\ttfamily system(\char`\"{}pause\char`\"{})} 函数说明 

\begin{DoxyNote}{注解}
C语言中system(\char`\"{}pause\char`\"{})是什么作用和意思 \href{https://blog.csdn.net/haiross/article/details/45093351}{\texttt{ https\+://blog.\+csdn.\+net/haiross/article/details/45093351}} ~\newline
system 就是调用从程序中调用系统命令（和 shell 命令）。 system(\char`\"{}pause\char`\"{})就是从程序里调用“pause”命令; 而“pause”这个系统命令的功能很简单， 就是在命令行上输出一行类似于“\+Press any key to exit”的字，等待用户按一个键，然后返回。 ~\newline
linux下运行程序出现“sh\+: 1\+: pause\+: not found” \href{https://www.cnblogs.com/feifanrensheng/articles/8696007.html}{\texttt{ https\+://www.\+cnblogs.\+com/feifanrensheng/articles/8696007.\+html}} ~\newline
一般在windows平台写代码为了在终端看到运行结果，所以加入了 {\ttfamily system(\char`\"{}pause\char`\"{});} 语句。 但是在linux下shell里pause不再是一条命令，因此会出现sh\+: 1\+: pause\+: not found的提示。 打开源文件删除system(\char`\"{}pause\char`\"{});语句即可。 另外在用windows上\+VS编写的代码，在linux下运行必须先转码才行，否则会出现类似在ubuntu/linux系统下system是包含在stdlib.h头文件中的 要实现类似功能的方法，要么编程system(\char`\"{}read\char`\"{}); 要么不用system命令，直接getchar()；不过跟system(\char`\"{}pause\char`\"{})的区别是，后者是按任意键继续，但是前面两种方法都是按回车键继续的。 
\end{DoxyNote}
