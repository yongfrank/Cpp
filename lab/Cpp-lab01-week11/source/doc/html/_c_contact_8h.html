string::npos 静态成员常量

是对类型为 size_t 的元素具有最大可能的值。 当这个值在字符串成员函数中的长度或者子长度被使用时，该值表示“直到字符串结尾”。 作为返回值他通常被用作表明没有匹配。

if (s1.find(s2) != std::string::npos) {
std::cout << "found!" << '\n';
}
